# 🦁 Ponira Lab.

> **Onde a alma do design encontra o corpo da tecnologia.**

Agência boutique de design e desenvolvimento focada em rituais digitais imersivos e performance bruta.

## 🛠️ Stack Tecnológica

- **Framework:** Next.js 16 (Turbopack)
- **Estilização:** Tailwind CSS (v4)
- **Tipografia:** Stelka Ultra & Archivo Variable
- **Efeitos:** React Bits (Grainient) & Framer Motion

## 📁 Estrutura

- `/public/fonts`: Ativos tipográficos locais.
- `/components/ui`: Componentes de interface e backgrounds complexos.
